Как исправить

Ваше OAuth-приложение на Яндексе не имеет нужных разрешений (scopes).

  1. Откройте oauth.yandex.com
  2. Выберите ваше приложение (client_id: {{ settings.yandex_client_id[:8] }}...)
  3. Перейдите в раздел "Доступы" (Permissions)
  4. Включите галочками нужные разрешения:
    • Yandex.Direct API
    • Yandex.Metrika API
    • Yandex.Webmaster API
    • Yandex.Audience API
    • AppMetrica API
  5. Убедитесь, что в Redirect URI указан: https://app.mais.agency/admin/oauth/callback
  6. Сохраните изменения и попробуйте снова

После изменения настроек может потребоваться 5-10 минут, чтобы изменения вступили в силу.
